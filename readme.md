1. Projekt clonen
2. .env.template kopieren
3. cp .env.template .env (Linux) oder copy .env.template .env (Windows, CMD) oder Copy-Item -Path .env.template -Destination .env (Windows, PowerShell)
4. Docker Desktop installieren bzw. öffnen
5. docker-compose up --build
6. docker-compose down (fährt Container herunter)
